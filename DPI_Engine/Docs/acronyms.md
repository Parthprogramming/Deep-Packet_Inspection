**SNI** - Server Name Identification 
**PORT_SCAN** -  Source IP contacted an unusually high number of distinct destination ports
**UNKNOWN_TLS** - HTTPS Connection has no SNI
**BEACONING** -  High packet count with near-symmetrical forward/backward ratio. Pattern matches C2 beaconing or malware keep-alive.
**C2** - Command & Control
**DNS_TUNNELING** - DNS packet is abnormally large. Could be encoding data inside DNS to evade firewall inspection
**Pattern** matches C2 beaconing
**TLS** - Transport layer Security
**TCP** - Transport Control Protocol
**UDP** - User Datagram Protocol
**ICMP** - Internet Control Message Protocol
